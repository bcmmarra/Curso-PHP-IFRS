1.1.2 Evolução
ECMAScript é a especificação padronizada para a linguagem de script que serve como base para JavaScript. A especificação define como a linguagem deve se comportar, suas funcionalidades e recursos (MDN Web Docs, 2024; FLANAGAN, 2014; GRONER, 2018). Desde sua primeira padronização, o ECMAScript passou por diversas revisões visando adicionar funcionalidades, aprimorar eficiência e fortalecer a segurança. Um marco significativo foi alcançado com o lançamento do ECMAScript 3 em 1999, que introduziu muitos dos recursos fundamentais ainda amplamente utilizados hoje.  O ECMAScript 3 introduziu a capacidade de tratar exceções usando blocos try, catch e finally, que permite que os desenvolvedores lidem com erros de forma mais elegante e controlada. Também trouxe métodos adicionais para strings, como: replace(), split(), slice(), substr(), e charAt(). O ECMAScript 3 também trouxe suporte para expressões regulares, permitindo busca e manipulação avançada de texto. Embora o JavaScript Object Notation (JSON) como um formato tenha sido popularizado mais tarde, a estrutura de objetos introduzida no ECMAScript 3 pavimentou o caminho para o uso extensivo de JSON para armazenamento e troca de dados. Após um período de desenvolvimento mais lento e dificuldades com o ECMAScript 4, o ECMAScript 5 foi lançado em 2009, trazendo melhorias substanciais em compatibilidade e segurança. 

Uma grande transformação veio com o ECMAScript 6 (também conhecido como ECMAScript 2015 ou ES6), que trouxe uma série de novidades, incluindo classes, módulos, promessas, bem como a introdução das declarações let e const, entre outros aprimoramentos. Essas adições modernizaram a linguagem e fortaleceram seu suporte para o desenvolvimento de aplicações web complexas. A partir de ECMAScript 7 (ECMAScript  2016), a Ecma começou a liberar atualizações anuais ao ECMAScript, cada uma adicionando novas funcionalidades e melhoria.

Atualmente, o JavaScript não se limita mais aos navegadores. Desde a introdução do Node.js em 2009, o JavaScript expandiu-se para o desenvolvimento de servidores, e frameworks populares como React, Angular e Vue.js capacitaram o desenvolvimento de interfaces front-end robustas e dinâmicas. Assim, JavaScript se tornou uma das bases do desenvolvimento web moderno, sendo utilizado em uma variedade de aplicações, desde páginas web interativas até o desenvolvimento de servidores completos e aplicativos móveis.

A história do JavaScript é marcada por uma rápida inovação e ampla adoção, consolidando sua posição como uma das linguagens de programação mais importantes e influentes no mundo da tecnologia.
