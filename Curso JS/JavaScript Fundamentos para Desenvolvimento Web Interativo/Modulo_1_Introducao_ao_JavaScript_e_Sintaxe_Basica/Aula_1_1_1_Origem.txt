1.1.1 Origem
O JavaScript surgiu em 1995, criado por Brendan Eich durante seu período na Netscape Communications Corporation. Naquela época, a internet estava emergindo como uma força comercial significativa, com a Netscape estabelecendo-se como o principal navegador. O propósito fundamental do JavaScript era revolucionar a experiência online, introduzindo interatividade nos sites que, até então, eram predominantemente estáticos (MDN Web Docs, 2024; W3C Schools, 2024). Essa inovação visava proporcionar aos usuários uma experiência mais dinâmica e envolvente na web.

O projeto original recebeu o nome de Mocha, e Eich conseguiu desenvolvê-lo em um incrível período de apenas 10 dias. Esta conquista é ainda mais notável quando consideramos a influência duradoura que a linguagem teve. Posteriormente, ela passou a ser conhecida como LiveScript. Mais adiante, ainda em 1995, a Netscape fez uma parceria com a Sun Microsystems, que levou a essa última mudança de nome: JavaScript. Essa mudança de nome refletia uma estratégia para capitalizar a crescente popularidade do Java, uma linguagem de programação desenvolvida pela Sun Microsystems na mesma época. Apesar das semelhanças nos nomes, é importante destacar que Java e JavaScript são linguagens muito distintas em termos de design e propósito.

No final de 1995, o JavaScript foi incorporado ao navegador Netscape Navigator. Essa integração em um navegador de grande destaque ajudou a consolidar o JavaScript como um elemento essencial da web. Seguindo o exemplo, o Microsoft Internet Explorer também implementou suporte ao JavaScript para competir com o Netscape, o que desempenhou um papel crucial na consolidação da posição da linguagem como uma tecnologia padrão na web.

Para assegurar a compatibilidade entre diversos navegadores, o JavaScript foi submetido à Ecma International, uma organização dedicada à padronização de tecnologias de informação e comunicação, que desempenhou um papel crucial no desenvolvimento de uma especificação comum. Isso resultou no surgimento do ECMAScript em 1997, a especificação oficial que estabelece os fundamentos e padrões da linguagem JavaScript. O ECMAScript define a sintaxe, os tipos, as operações e os objetos da linguagem, garantindo a consistência na execução do código JavaScript em várias plataformas e navegadores.
